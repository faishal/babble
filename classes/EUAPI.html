filter_plugins_api()

filter_plugins_api(bool|object $default, string $action, object $plugin) : bool|\WP_Error|\EUAPI_Info

Before the Plugin API performs an action, this short-circuit callback is fired, allowing us to override the API method for a given action.

Here, we override the action which fetches plugin information from the wp.org API and return our own plugin information if necessary.

Parameters

bool|object $default

Default return value for this request. Usually boolean false.
string $action

API function being performed.
object $plugin

Plugin Info API object.

Returns

bool|\WP_Error|\EUAPI_Info —

EUAPI Info object, WP_Error object on failure, $default if we're not interfering.
